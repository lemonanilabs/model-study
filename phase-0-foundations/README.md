# Phase 0 — 基礎數學與程式能力

## 目標
看到公式不害怕，能用程式碼驗證每個數學概念。

## 檔案順序

| 順序 | 檔案 | 主題 | 預計時間 |
|------|------|------|----------|
| 1 | `01_numpy_basics.py` | NumPy 矩陣運算基礎 | 1-2 天 |
| 2 | `02_linear_algebra.py` | 線性代數：向量、矩陣、變換 | 2-3 天 |
| 3 | `03_calculus.py` | 微積分：導數、梯度、Chain Rule | 2-3 天 |
| 4 | `04_probability.py` | 機率統計：分佈、貝氏、MLE | 2-3 天 |
| 5 | `05_matplotlib_basics.py` | 資料視覺化基礎 | 1 天 |

## 使用方式

每個檔案都可以直接執行：
```bash
python 01_numpy_basics.py
```

建議邊看程式碼邊在 Python REPL 或 Jupyter 中自己試。
動手改數字、改維度，觀察輸出的變化，這比死背公式有效 100 倍。
